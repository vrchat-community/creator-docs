# Roadmap

This page gives you an overview of upcoming and recently completed features for avatar and world creators.

:::note 

This page was last updated on March 23, 2025 and may be outdated. For the latest information, visit VRChat's [feedback board](https://feedback.vrchat.com/),  [forum](https://ask.vrchat.com/c/official/31), and [SDK release notes](/releases/).

:::

Please keep in mind that this list is subject to change. It also doesn’t include:

[^1]: This feature is available in VRChat, but may require VRChat's latest open beta SDK.
[^2]: "Open Beta" means that any VRChat user can test this feature by downloading a beta version of VRChat. We appreciate your feedback and bugs reports!
[^3]: "In Progress" means that the feature is getting active development work! We are building the feature and performing internal testing and feedback cycles.
[^4]: "Planned" means that we haven't started major development on the feature, but we plan on working on it. The feature may require some design and scoping work.
[^5]: This URL leads to a private feedback board for creators participating in the Creator Economy.
